# План реализации задачи MM-201: Main Menu UI Overhaul

## Описание задачи
Обновить макет в `screen.rs` для AAA эстетики (типографика, отступы) и добавить плавные анимации наведения (масштабирование/цветовые сдвиги).

## Анализ текущего состояния
- Главное меню реализовано в `src/launcher/menu/screen.rs`
- Используются темы из `src/ui/theme/` с цветами, шрифтами и размерами
- Кнопки уже имеют базовое изменение цвета при наведении в `src/launcher/menu/widgets/buttons.rs`
- Макет задается в `src/launcher/menu/layout.rs`

## Подробный план реализации

### 1. Улучшение типографики
- Увеличить размер шрифта заголовка меню с 48px до 64px для лучшей читаемости
- Проверить контрастность текста с фоном согласно стандартам доступности WCAG AAA
- Убедиться, что толщина шрифта остается bold для лучшей видимости

### 2. Оптимизация отступов и расстояний
- Увеличить отступы внутри панели меню с 16px до 24px
- Увеличить отступ под заголовком с 40px до 60px для лучшего разделения элементов
- Изменить ширину панели меню с 400px на 500px для более современного вида
- Увеличить высоту кнопок с 50px до 60px для лучшей кликабельности

### 3. Добавление анимации масштабирования
- Реализовать плавное увеличение кнопок при наведении курсора (scale 1.0 -> 1.05)
- Добавить систему анимации, которая будет обрабатывать изменение трансформации кнопок
- Обеспечить плавность анимации с использованием подходящей длительности (около 0.2 секунд)

### 4. Улучшение цветовых переходов
- Добавить плавные переходы между цветами фона кнопок при наведении
- Улучшить визуальную обратную связь при взаимодействии с кнопками
- Обеспечить достаточный контраст между обычным и активным состоянием кнопок

### 5. Тестирование и проверка
- Протестировать изменения на разных разрешениях экрана
- Проверить доступность интерфейса
- Убедиться, что анимации не влияют отрицательно на производительность

## Технические детали реализации

### Файлы для модификации:
1. `src/launcher/menu/screen.rs` - основной файл для обновления макета
2. `src/launcher/menu/layout.rs` - обновление констант макета
3. `src/launcher/menu/widgets/buttons.rs` - добавление анимации наведения
4. `src/ui/theme/metrics.rs` - возможно обновление размеров по умолчанию

### Предлагаемые изменения:
1. Для типографики:
   - Изменить `theme.sizes.font_h1` с 48.0 на 64.0 в функции `spawn_title`
   - Увеличить `TITLE_MARGIN_BOTTOM` в layout.rs с 40.0 до 60.0

2. Для размеров панели:
   - Изменить `PANEL_WIDTH` в layout.rs с 400.0 на 500.0
   - Изменить внутренние отступы с 16.0 на 24.0

3. Для анимации наведения:
   - Добавить компонент Transform к кнопкам
   - Создать систему анимации, которая будет изменять масштаб при наведении
   - Использовать интерполяцию для плавного изменения масштаба

## Критерии успеха
- Заголовок меню стал более заметным и читаемым
- Интерфейс получил более современный и профессиональный вид
- Анимации наведения плавные и приятные для пользователя
- Все изменения соответствуют принципам доступности
- Не наблюдается просадки производительности
